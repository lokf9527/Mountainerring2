## 網站功能

* 電商展示：首頁、商品列表、商品介紹
* 購物車：刪除、修改商品數量、優惠券、結帳
* 商品收藏功能：收藏列表、刪除
* 文章專欄：文章列表、文章內容

## 使用技術

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
