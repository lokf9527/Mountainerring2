## 網站功能

* 電商展示：首頁、商品列表、商品介紹
* 購物車：刪除、修改商品數量、優惠券、結帳
* 商品收藏功能：收藏列表、刪除
* 文章專欄：文章列表、文章內容

## 使用技術

* `Vue 3 Options API`
* `Vite` 環境建置
* 引入 `Vue Router` 建構路由
* 引入 `Vue Axios` 串接 `RESTful API` 資料
* 引入 `Vee Validation` 結帳表單驗證
* 透過 `Pinia` 傳遞跨元件資料
* 使用 `localStorage` 製作商品收藏功能
* Layout 使用 `Bootstrap 5` 結合 `SCSS` 切版
* icon 使用 `Bootstrap icons`
* 開發後台管理系統
